# Task-Management-System



The Task Management System is a simple and efficient SQL-based project designed to organize, track, and manage tasks in a structured way. It uses relational database concepts to store task information, update statuses, categorize priorities, and retrieve records effectively.
This project demonstrates core SQL operations such as table creation, relationships, constraints, and CRUD functionalities, showing how a database can be used to support real-world task management workflows.
It is ideal for beginners learning SQL or anyone looking for a clear example of database-driven task organization.
